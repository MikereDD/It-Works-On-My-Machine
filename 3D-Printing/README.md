# 3D-Printing

Projects, experiments, prototypes, and functional nonsense created for the Bambu Lab X2D.

## About

This directory contains 3D printing projects designed, modified, or tested on the Bambu Lab X2D.

Some things are practical.  
Some things are sarcastic.  
Some things exist because the printer was idle for five minutes.

You may find:
- OpenSCAD projects
- STL files

- Desk organization prints
- Utility parts
- Random experiments
- Prototype designs
- “Why did I make this?” moments

Primary printer:

## Philosophy

If it:
- solves a problem,
- organizes chaos,
- improves workflow,
- makes me laugh,
- or should not physically exist...

it belongs here.

## Notes

Most projects are designed around:
- easy printing
- low support usage
- functional reliability
- fast iteration

Many designs will evolve over time as better ideas replace older dumb ones.
