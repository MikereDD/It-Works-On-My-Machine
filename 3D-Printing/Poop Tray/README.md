# Sharting Tray

A highly advanced waste-management platform engineered with cutting-edge sarcasm and absolutely no dignity.

- Universal desktop junk containment
- Snack crumb retention technology
- Screws, adapters, USB nonsense, and random parts holder
- Emergency “where did I put that?” zone

- 3D printable on basically anything

## Recommended Settings

### PLA / PLA+
- Layer Height: `0.20`
- Walls: `3`
- Infill: `15%`
- Supports: `No`
- Brim: Optional

### Nylon
If you hate yourself professionally:
- Dry filament first
- Slow down
- Use enclosure
- Pray to the warp gods
