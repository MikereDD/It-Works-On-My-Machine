# 🖨️ Bambu X2D Riser v4

> More height. More storage. More unnecessary engineering.

---

## 🧠 Overview

This project is a custom riser system for the **Bambu Lab X2D** printer.

Designed to:

- raise the printer
- add functional storage
- improve workspace organization
- provide modular expansion points

Built with OpenSCAD and exported as printable STL components.

---

## 📦 Included Files

### OpenSCAD Source

```text
bambu_x2d_riser_v4.scad
```

Primary editable source file for the riser system.

## 🖨️ Printing Notes

Recommended:

- PLA+
- PETG for higher temperature resistance
- 0.2 layer height
- 3–4 walls
- 15–25% infill

Large components may require:

- slower speeds
- brim support
- larger print area orientation adjustments
