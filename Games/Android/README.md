# 📱 Games / Android

> Tiny games. Bad ideas. Touch controls.

---

## 🧠 Overview

This directory contains Android game projects, APK experiments, and mobile ports of terminal-style games.

Most projects here are:

* Lightweight
* Experimental
* Built for fun
* Probably unfinished

---

## 🎮 Current Direction

Planned projects include:

* Tetris-style games
* Arcade clones
* Puzzle games
* Minimal retro-style UI
* Terminal-inspired interfaces

---

## ⚙️ Goals

The goal is not to compete with modern mobile games.

The goal is:

* small size
* fast loading
* simple controls
* offline play
* fun mechanics

---

## 🛠️ Development

Primary tooling:

```text id="0j0z7m"

Gradle
```

* Godot

## 📦 APKs

Release APKs may eventually be stored here or linked externally.

No guarantees:

* compatibility
* stability
* battery life
* sane code
