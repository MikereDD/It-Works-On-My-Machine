# 🧱 TetBlockRis v1.0

> Falling blocks. Poor decisions. Retro chaos.

---

## 🧠 Overview

TetBlockRis is a lightweight Tetris-style game project focused on:

* fast gameplay
* simple visuals
* arcade feel
* minimal bloat

Built as part of the `Games/Android` section of the repository.

---

## 🎮 Features

### Current

* Falling block gameplay
* Line clearing
* Score tracking
* Keyboard / touch controls
* Retro-inspired visuals

---

## ⚙️ Design Goals

TetBlockRis is intentionally simple.

The goal is:

* instant gameplay
* low overhead
* responsive controls
* no garbage mobile monetization

No ads.
No battle pass.
No login screen.

Just blocks.
