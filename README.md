# 🧰 It-Works-On-My-Machine

> Everything in here works. On my machine.

---

## 📦 What this is

A personal archive of scripts, projects, experiments, and random things I’ve built and didn’t want to lose.

* **Windows** → PowerShell
* **Linux** → Bash
* **Bots** → automation tools and APIs
* **Games** → Android, Windows, Linux, and terminal games
* **3D Printing** → functional prints, OpenSCAD projects, and prototypes

If it’s here, it probably worked once. That’s the bar.

---

## 🪟 Windows / PowerShell

PowerShell scripts and utilities for Windows systems.

Includes:

- Admin tools
- Personal tools
- Media workflows
- Automation scripts
- System utilities
- Menu-driven tool launchers

Most scripts are designed around my own environment and workflow.

---

## 🐧 Linux / Bash

Bash scripts and Linux utilities for servers, Raspberry Pi systems, and automation.

Includes:

- System setup
- Maintenance scripts
- Network utilities
- Bot hosting
- tmux workflows
- Arch Linux experiments

Most Linux tooling is developed around Arch Linux and Raspberry Pi systems.

## 🎮 Games

Game projects, terminal experiments, and small arcade-style projects.

Includes:

- PowerShell terminal games
- Bash terminal games
- Retro-style experiments

Some games are playable. Some are prototypes. Some exist because boredom is dangerous.

---

## 🤖 Bots

Some scripts in this repo are simple bots or automation tools.

* Mostly built for personal use
* May rely on APIs (Telegram, OpenAI, etc.)
* Config is not included (tokens, keys, IDs)

They are not designed to be deployed cleanly or run by others without modification.

If you want to use them:

* read the script
* figure out what it expects
* supply your own config

## 📁 Structure

```text
Windows/       → PowerShell scripts
Linux/         → Bash scripts
Bots/          → automation scripts (Telegram, API-based, etc.)
Games/         → Android, Windows, Linux, and terminal games
3D-Printing/   → OpenSCAD projects, STL files, prototypes
```

Each side is independent. No attempt is made to keep them in sync.
