# 🧰 It-Works-On-My-Machine

> Everything in here works.
> On my machine.

---

## 📦 What this is

A personal archive of scripts I’ve used at some point and didn’t want to lose.

* **Windows** → PowerShell
* **Linux** → Bash
* **Bots** → small automation tools (Telegram, API-based, etc.)

If it’s here, it probably worked once. That’s the bar.

---

## 🤖 Bots

Some scripts in this repo are simple bots or automation tools.

* Mostly built for personal use
* May rely on APIs (Telegram, OpenAI, etc.)
* Config is not included (tokens, keys, IDs)

They are not designed to be deployed cleanly or run by others without modification.

If you want to use them:

* read the script
* figure out what it expects
* supply your own config

---

## ❌ What this is not

* Not a polished project
* Not a library
* Not designed for general use
* Not guaranteed to work anywhere else

---

## 📁 Structure

```text
Windows/   → PowerShell scripts
```
