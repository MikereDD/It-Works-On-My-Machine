# v5.8.3 — Queue Message Cleanup & DM UX Polish

## Overview

v5.8.3 focuses on queue message cleanup behavior and overall Telegram
chat polish.

This release improves the upload experience by ensuring temporary queue
confirmation messages are automatically removed after processing begins,
including inside direct messages.

The result is a significantly cleaner and more polished Telegram UX.

---

## Fixed

### 🧹 Queue Message Cleanup in DMs

Raziel now properly deletes temporary:
- “Added to queue”
- queue confirmation
- enqueue status

messages after upload processing begins in:
- private chats
- direct messages
- group chats
- operational channels

Previously, some DM queue messages could remain visible after uploads
completed.

---

## Improved

### 📥 Queue Tracking Behavior

Queue confirmation messages are now:
- tracked consistently
- attached directly to queued jobs
- cleaned up uniformly across all chat types

This improves:
- upload lifecycle consistency
- queue state management
- message cleanup reliability
