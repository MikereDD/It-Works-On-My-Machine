# v5.9 — Config-Controlled Validation Policy

## Overview

v5.9 redesigns Raziel’s validation architecture by introducing a
config-controlled validation policy system.

This release removes the need to continuously expand hardcoded platform
presets whenever yt-dlp adds support for new websites.

Raziel can now operate in either:
- strict allow-list mode
- open yt-dlp mode

directly from configuration.
