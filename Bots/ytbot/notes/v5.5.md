# v5.5 — Runtime Process Restart Control

## Overview

v5.5 introduces safe runtime restart control for Raziel.

Administrators can now fully restart the bot directly from Telegram
without manually touching tmux, shell sessions, or system services.

Raziel now supports:
- self-managed process restarts
- tmux-safe runtime replacement
- operational control directly from Telegram
- safer live development workflows

while preserving:
- tmux session state
- shell environment
- virtual environment execution

---

## Added

### 🔄 `/restart`

Admin-only restart command.

/restart
