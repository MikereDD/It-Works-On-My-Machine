# YTBot v4.2

## Overview
Expansion into a full media pipeline with automation and routing.

## Changes
- added metadata extraction (title, duration, uploader)
- implemented file routing:
  - done/video
  - done/audio
  - done/failed
- added optional archive forwarding (ARCHIVE_CHAT_ID)
- added watch folder ingestion
- added CLI mode:
  - --url
  - --audio
- enhanced /stats with:
  - unique users
  - domain tracking (top sites)

## Notes
This version expands the bot beyond Telegram input into a multi-source pipeline system.

## Impact
Introduces automation and turns the bot into a reusable media processing tool.
