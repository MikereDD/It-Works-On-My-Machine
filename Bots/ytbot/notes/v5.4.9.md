# v5.4.9 — Runtime Configuration Reloading

## Overview

v5.4.9 introduces live runtime configuration reloading for Raziel.

Administrators can now reload `ytbotrc.py` without restarting the bot,
allowing rapid iteration and safer operational tuning during active use.

This improves:
- development workflow
- live testing
- production adjustments
- platform management
- debugging

while preserving:
- queue state
- active workers
- running uploads
- persistent history
