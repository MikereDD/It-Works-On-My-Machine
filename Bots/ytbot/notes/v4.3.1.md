# YTBot v4.3.1

## Overview
Stability and config alignment update following v4.3.

## Changes
- expanded ytbotrc template to include all supported options
- added access control settings (ADMIN_USERS, ALLOWED_USERS, ALLOW_ALL_USERS)
- added behavior settings (WATCH_FOLDER_ENABLED, DOWNLOAD_TIMEOUT)
- added optional ARCHIVE_CHAT_ID
- added optional BASE_DIR override
- improved config documentation and safety

## Notes
This version focuses on making configuration clearer and preventing ambiguity between repo and live runtime setup.

## Impact
No major feature changes, but improves maintainability and clarity.
