# v5.2

## Summary
Fixes group chat monitoring for shared and forwarded links.

## Changes
- Removed dependency on `message.text` only
- Added support for:
  - captions
  - Telegram entities (`url`, `text_link`)
- Improved URL extraction logic for shared previews
- Updated handler to process both TEXT and CAPTION messages

## Result
Bot now correctly:
- detects pasted URLs
- detects shared links (preview cards)
- detects forwarded messages with links
- auto-queues downloads in allowed group chats

## Notes
No changes to:
- queue system
- commands (/dl, /audio, /clip)
- archive handling
- UI system

## Version
5.2
