# Raziel / YTBot v6.1

## Overview

v6.1 introduces platform-aware visual branding and cleaner Telegram
caption presentation.

Building on the metadata-aware upload architecture introduced in v6.0,
Raziel now visually identifies supported media platforms using
platform-specific icons and cleaner upload formatting.

This improves:

* upload readability
* platform recognition
* Telegram scanning usability
* archival presentation quality
* visual consistency across uploads

---

## Platform-Aware Upload Branding

### New Platform Icon System

Raziel now automatically applies platform-specific branding icons
inside upload captions.

Supported platform displays:

```text
▶ YouTube
𝕏 Twitter
◎ Instagram
♫ TikTok
ⓕ Facebook
⬡ Reddit
◉ BitChute
```

This replaces the older generic:

```text
🌐 Platform
```

style presentation.

---

## Cleaner Telegram Caption Layout

### Improved Media Presentation

Uploads now feel more structured and visually distinct while preserving
Raziel's lightweight operational design.

Example:

```text
🎞️ Media Title

Source text...

◉ Uploader
𝕏 Twitter
◷ 1m 37s

⤷ Source URL
```

Benefits:

* faster visual scanning
* easier source identification
* reduced emoji clutter
* cleaner dark-theme appearance
* more polished upload presentation

---

## Platform Display Refinement

### X / Twitter Rendering Fix

Adjusted platform rendering logic to avoid duplicated or awkward
Telegram font behavior such as:

```text
X X / Twitter
```

Raziel now intelligently formats X/Twitter as:

```text
𝕏 Twitter
```

for cleaner cross-device rendering consistency.
