# v5.3

## Summary
Major upgrade enabling large file uploads using local Telegram Bot API.

## Changes
- Fixed critical bug in `build_app()` (aapp → app)
- Enabled local Bot API server support
- Added:
  - base_url = http://127.0.0.1:8081/bot
  - base_file_url = http://127.0.0.1:8081/file/bot
- Increased upload limit from 49MB → ~1.9GB
- Replaced hardcoded compression trigger with MAX_UPLOAD_BYTES
- Preserved shared/forwarded link detection
- Maintained full queue system and command support

## Result
Bot now:
- uploads large videos (podcasts, long content)
- avoids aggressive compression
- avoids multi-part splitting
- preserves video quality

## Requirements
Local Telegram Bot API server must be running:
http://127.0.0.1:8081
