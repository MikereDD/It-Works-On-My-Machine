# v5.6 — Live Progress System

## Overview

v5.6 introduces Raziel’s first real-time live activity system.

Raziel can now continuously update Telegram status messages during:
- downloads
- processing
- media transfers

using live progress editing instead of static status messages.

This makes Raziel feel significantly more responsive and operationally
transparent during large downloads and long-running jobs.

---

## Added

### 📊 Live Download Progress

Raziel now displays real-time yt-dlp progress updates directly inside Telegram.

Live status messages now include:
- progress bars
- percentage completion
- download size
- transfer speed
- ETA estimates

🎞️ Example Video
