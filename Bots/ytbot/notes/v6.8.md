# Raziel v6.8 — Metadata-On-Demand Command System

## Overview

Raziel v6.8 redesigns source-context delivery into an explicit opt-in workflow.

Metadata and expandable source-context posts are no longer sent by default.

Instead, Raziel now follows a:

```text
quiet by default
verbose when explicitly requested
```

architecture.

This significantly reduces Telegram group noise while preserving advanced source-context functionality for users who actually want it.

---

# 🧠 Core UX Change

## Previous Behavior

Before v6.8:

```text
/dl
→ media upload
→ metadata/context reply
```

This created a two-post workflow:

1. media upload
2. expandable metadata/context post

While useful at times, it added unnecessary clutter in active groups.

---

# ✨ New Behavior

## Default Commands = Quiet

Normal commands now only post the media upload itself.

Examples:

```text
/dl
/hd
/full
/audio

/rdl
/rhd
/rfull
/raudio
```

Behavior:

```text
download media
→ clean caption
→ done
```

No expandable metadata/context reply is sent.

---

# 📖 Metadata Commands

Extended source-context behavior is now explicit.

New commands:

```text
/dlmeta
/hdmeta
/fullmeta
/audiometa

/rdlmeta
/rhdmeta
/rfullmeta
/raudiometa
```

These commands:

* enable expandable source-context replies
* preserve uploader/post metadata
* preserve long-form extracted descriptions
* preserve source commentary/context

Behavior:

```text
download media
→ clean upload
→ expandable source-context reply
```

---

# 🧩 Internal Architecture

v6.8 introduces:

```python
include_metadata = True
```

as a per-job behavior flag.

Default jobs:

```python
include_metadata = False
```

Metadata commands:

```python
include_metadata = True
```

Expandable context generation is now conditional instead of global.

---

# 🧹 Telegram UX Improvements

v6.8 further improves:

* low-noise Telegram groups
* media-first workflows
* passive watcher cleanliness
* source-link handling
* command clarity
* intentional verbosity

This aligns Raziel with its evolving philosophy:

```text
download media
present it cleanly
stay quiet unless asked
```

# 🎯 Design Direction

Raziel now supports two operational styles:

## Quiet Mode (default)

```text
simple
fast
clean
minimal
```

## Metadata Mode

```text
verbose
source-aware
context-rich
archival
```

Users choose the workflow intentionally through command selection.
