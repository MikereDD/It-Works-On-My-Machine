# Raziel / YTBot v6.0

## Overview

v6.0 introduces metadata-aware uploads and self-contained media archiving.

Raziel now preserves source context from supported platforms and attaches
cleaned post descriptions, captions, tweet text, uploader information,
platform labels, and source links directly to uploaded Telegram media.

This transforms Raziel from a simple download/upload bot into a richer
media archival and context-preserving platform.

---

## Metadata-Aware Upload Captions

### Full Source Context Preservation

Raziel now attempts to preserve:

* X/Twitter post text
* YouTube descriptions
* Instagram captions
* TikTok captions
* Reddit post text
* Facebook descriptions (when exposed by extractors)
* BitChute descriptions

Uploaded media now retains the original source context instead of only
showing minimal file information.

---

## Unified Caption Pipeline

### New Caption Builder System

Introduced:

```python
build_upload_caption(meta, mode)
```

This creates consistent Telegram captions across all supported platforms.

Raziel now intelligently formats:

* media title
* source text/about text
* uploader attribution
* source platform
* duration
* clip range
* original source URL

---

## Cleaner Telegram Upload Presentation

### Improved Upload Formatting

Uploads now follow a cleaner archival layout:

```text
🎞️ Media Title

Original source text...

👤 Uploader
🌐 Platform
⏱️ Duration

🔗 Source URL
```

This creates:

* self-contained uploads
* cleaner forwards
* improved Telegram searchability
* better media provenance tracking
* stronger long-term archival quality

---

## Metadata Processing Improvements

### Metadata Cleaning

Added:

```python
clean_metadata_text()
```

This improves:

* paragraph spacing
* newline handling
* whitespace cleanup
* long text trimming
* caption readability

### Smart Caption Trimming

Raziel now intelligently trims oversized descriptions to remain within
Telegram caption limits while preserving the most useful content.

---

### Automatic Platform Labels

Raziel now automatically labels uploads by platform:

* YouTube

* Instagram
* Facebook

* Reddit
* BitChute

This improves media organization and readability.
