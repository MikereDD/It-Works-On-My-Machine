# v5.8.4 — Facebook Platform Registry Fix

## Overview

v5.8.4 fixes Facebook platform validation support inside Raziel’s
video-source registry layer.

This release corrects a platform preset inconsistency where Facebook
could be enabled in configuration while still being rejected by the
validation system.

---

## Fixed

### 📘 Facebook Validation Support

Raziel now properly recognizes Facebook video URLs during source
validation.

Supported domains now include:

```python
"facebook": (
    "facebook.com",
    "m.facebook.com",
    "www.facebook.com",
    "fb.watch",
),

---
