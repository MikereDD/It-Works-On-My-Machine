# v5.8.2 — Telegram Inline Mode & Global Chat Access

## Overview

v5.8.2 introduces Telegram Inline Mode support for Raziel.

Raziel can now respond from chats and groups where the bot itself is
not a member by using Telegram’s inline query system.

This expands Raziel from:
- group-bound interaction
into:
- globally accessible Telegram utilities

while preserving the existing:
- mention command layer
- operational chat behavior
- media pipeline architecture
