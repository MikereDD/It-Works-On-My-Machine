# Raziel v6.9 — Quiet-First Metadata Architecture

## Overview

Raziel v6.9 refines the Telegram UX philosophy around metadata, captions, and group noise.

The previous expandable metadata/context system worked technically, but produced unnecessary multi-message noise in active chats.

v6.9 changes the architecture to:

```text
default = quiet
metadata = explicit opt-in
```

This creates a significantly cleaner Telegram-native workflow while still preserving rich source context when users intentionally request it.

---

# 🧠 Quiet-First Design

Raziel now defaults to:

```text
download media
send clean caption
done
```

Standard commands no longer produce extra source-context posts.
