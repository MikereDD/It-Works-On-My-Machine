# YTBot v4.3

## Overview
Major usability upgrade with interactive UI and system polish.

## Changes
- added /ui command with inline buttons:
  - Video
  - Audio
  - Cancel
- implemented callback-based job creation
- added smart format selection with fallback logic
- added ffmpeg-based compression for large files
- added Telegram upload fallback (video → document)
- added video validation using ffprobe
- improved error handling and failure routing
- improved logging and system feedback
- integrated weather and forecast commands

## Notes
This version focuses on user experience and reliability.

## Impact
Transitions the project from a backend tool into a user-friendly interactive system.
