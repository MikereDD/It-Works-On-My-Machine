## 🚀 Overview

Zaphkiel is a Telegram AI assistant built around OpenAI models, conversational workflows, image generation, and image transformation.

Designed to be a practical personal AI assistant rather than a general-purpose chatbot.

---

## Features

- GPT-powered conversations
- Image generation
- Image-to-image transformations
- Style presets
- Telegram integration
- Local Bot API support
- Admin controls
- Runtime configuration reloads
- Owner-only operation
