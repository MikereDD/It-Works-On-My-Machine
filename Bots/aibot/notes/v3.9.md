# Zaphkiel v3.9

## Overview

v3.9 focuses on image conversion workflows, style management, and overall user experience improvements.

This release refines image generation and image-to-image conversion while simplifying interaction through Telegram.

---

## New Features

### Style Management

Added support for reusable image style presets.

Examples include:

- Ghibli
- Anime
- Comic Book
- Pencil Sketch
- Watercolor

Styles can be expanded and customized without modifying core image workflows.

### Image Conversion Workflow

Improved image conversion pipeline:

- Cleaner processing flow
- Better status reporting
- Simplified user interaction
- Consistent output handling

### Conversion Commands

```text
/convert
/styles
```

allow users to browse available styles and transform images directly from Telegram.

### Local Bot API Support

Continued optimization for Local Bot API deployments.

Benefits include:

- Faster media handling
- Reduced Telegram file limitations
- Improved reliability for large image workflows

### Administrative Controls

Administrative commands remain available:

```text
/reload
/restart
```

allowing configuration reloads and bot restarts without leaving Telegram.

---

## Improvements

### User Experience

- Cleaner status messages
- Reduced chat clutter
- More predictable conversion workflow
- Better error handling

### Style Processing

- Improved style consistency
- Better prompt handling
- More reliable image transformations

## Result

v3.9 further evolves Zaphkiel into a practical AI image assistant focused on reliable image generation, conversion, and experimentation directly from Telegram.
