# Sandalphon v3.2

## Overview

v3.2 focuses on metadata cleanup, title normalization, and improving the quality of Telegram audio presentation.

This release fixes cases where artist names could appear twice in audio titles when metadata from YouTube and other sources contained embedded artist information.

---

## Fixes

### Duplicate Artist Name Cleanup

Improved metadata processing to prevent artist names from appearing twice in Telegram audio cards.

Examples of previously problematic titles:

```text
Twenty One Pilots - twenty one pilots : Stressed Out
```

Now correctly displayed as:

```text
Twenty One Pilots - Stressed Out
```

### Improved Metadata Normalization

Added cleanup logic for titles that already contain artist information.

Supported formats include:

```text
Artist - Song
Artist: Song
Artist : Song
Artist – Song
Artist — Song
```

When artist metadata is already available, redundant artist prefixes are automatically removed from the title before display.

### Cleaner Telegram Audio Cards

Benefits include:

- Consistent Artist - Song formatting
- Improved readability
- Better library presentation
- Cleaner Telegram audio metadata

## Internal Changes

- Enhanced metadata cleanup pipeline
