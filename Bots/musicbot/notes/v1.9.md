# v1.9 — Spotify Metadata Matching

Released as part of the Sandalphon v1 series.

## Overview

v1.9 introduced Spotify metadata matching, allowing Sandalphon to use Spotify track information to improve search accuracy when locating music through YouTube and other yt-dlp-supported sources.

This release focused on metadata quality rather than audio sourcing.

---

## New Features

### Spotify Metadata Matching

Sandalphon can now extract metadata from Spotify track links, including:

- Artist name
- Track title
- Album information (when available)

This metadata is then used to perform more accurate searches through YouTube and other supported sources.

### Improved Match Accuracy

By using Spotify's structured track data, Sandalphon significantly improves the likelihood of locating the correct song, official release, or high-quality upload.

Benefits include:

- Better artist identification
- More accurate song matching
- Fewer incorrect search results
- Improved metadata consistency throughout the library

---

## Important Notes

Spotify audio is **not downloaded directly**.

Spotify links are used exclusively as a metadata source. Once metadata is collected, Sandalphon searches for matching audio using supported yt-dlp providers such as:

- YouTube
- SoundCloud
- Other supported sources

This approach maintains compatibility while benefiting from Spotify's high-quality track metadata.

---

## Result

v1.9 improved music discovery and matching quality while keeping Sandalphon's existing download workflow unchanged.
