# Sandalphon v1 Series

## Overview

The v1 series established the foundation of Sandalphon as a reliable Telegram music bot focused on downloading, tagging, caching, and delivering music through a simple chat interface.

Versions v1.0 through v1.7 introduced the core architecture that later releases expanded upon, including queue processing, metadata handling, Telegram integration, caching systems, and the overall user workflow.

---

## v1.0 → v1.7

Initial Sandalphon development and stabilization release series.

Key areas established during the v1 cycle:

- Telegram bot integration
- yt-dlp download workflow
- Audio processing pipeline
- Metadata extraction
- Queue-based processing
- Local caching foundation
- File cleanup and management
- Telegram-ready audio delivery
- Basic administration controls
- Performance and stability improvements
- General bug fixes and refinement

These releases formed the foundation for all future Sandalphon features introduced in the v2 and v3 series.
